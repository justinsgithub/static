# Vim Snips

> keys are represented between `< >` such as `<Enter>`

## command line mode 

### delete blank lines

```vim
:g/^\s*$/d 
```


## keys

### save and quit 

`<Z><Z>`

### fix spelling with dictionary

`<z><=>`

### fix spelling error with NUMBER dictionary item 

`<NUMBER><z><=>`

### bookmark spot to remember

`<m><m>`
